# 第七阶段：聊天核心开发计划

## 概述

ChatEditor 采用上中下布局：Header + MessageList + InputArea

## 功能清单

### 核心聊天功能

- 发送消息（支持文本和附件）
- 停止生成
- 重试消息
- 继续生成消息
- 编辑消息
- 编辑并重新发送消息
- 删除消息
- 消息星标/收藏

### 模型相关

- 模型选择器（ModelSelector）
- 为单条消息切换模型
- 支持多个 LLM 配置

### 消息树/分支功能

- 消息树侧边栏（MessageTreeSidebar）
- 分支导航器（BranchNavigator）
- 分支切换
- 路径管理（currentPath）
- 支持消息的树形结构（父子关系）

### 消息队列功能

- 消息队列面板（MessageQueuePanel）
- 队列添加/移除/编辑/重排序
- 自动处理队列
- 暂停/恢复队列
- AI 自动生成问题
- 导入提示词列表

### 消息展示

- 消息列表（MessageList）
- 消息折叠/展开（全部折叠、仅折叠 AI 消息）
- 消息搜索（MessageSearch）
- 流式内容显示
- 滚动到指定消息

### 收藏功能

- 添加消息到收藏夹
- 收藏文本片段
- 收藏夹文件夹管理

### 导出功能

- 导出模态框（ExportModal）
- 单条消息导出（SingleMessageExportContainer）

### 其他功能

- 引用文本到输入框
- 从选中文本创建新对话
- 页面溯源显示（PageLineageDisplay）
- 图片预览（ImagePreviewModal）
- 欢迎消息（WelcomeMessage）
- 文件附件支持

## MVP 路径

先跑通核心流程：

```
1.1 → 1.3 → 2.1~2.5 → 3.1~3.2
```

完成后可以：发送消息、显示流式响应、停止生成

## 依赖关系

```
messagesService
    ├── 依赖 pagesStore（消息属于页面）
    └── 依赖 settingsStore（获取 LLM/Model 配置）

useChat
    ├── 依赖 messagesService
    └── 依赖 aiService（流式响应）

ChatEditor
    ├── 依赖 useChat
    └── 依赖 useSettings（模型选择）
```
